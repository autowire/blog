# 王爵的一亩三分地

这是一个 `Jekyll` 主题的模板，我修改自 [theme-H2O](https://github.com/kaeyleo/jekyll-theme-H2O)，针对字体和板式以及功能做了大量优化。
我经常会用到一些灵活多变的功能，自定义比较强。所以选择了静态博客作为站点，如果你也喜欢它可以点个 `star`。

该主题是基于 `Jekyll3` + `Gulp` 构建的。

- `boudle install`
- `jekyll serve` 即可启动
